# 🚢 Final Submission Manifest

The Ajaia Collaborative Editor project is complete and ready for evaluation. This document lists exactly what is included in this submission.

## ✅ Project Scope
1. **Frontend**: React-based Single Page Application (SPA) with integrated rich-text editor (Quill).
2. **Backend**: Express-based Node.js API with JWT authentication and Socket.IO for real-time presence.
3. **Database**: MongoDB (Local or Atlas Atlas-ready).

- **Frontend URL**: [https://ajaia-docs-rho.vercel.app/](https://ajaia-docs-rho.vercel.app/)
- **Backend API**: [https://ajaia-backend.onrender.com/](https://ajaia-backend.onrender.com/)

## ✨ Key Features
- **User Authentication**: Secure JWT-based Login and Register.
- **Email Verification**: Brevo API integration for OTP signups.
- **Document Management**: Create, edit, and auto-save documents.
- **Real-time Presence**: Interactive "Active User" bubbles via Socket.IO.
- **Sharing & Collaboration**: Document sharing via email.
- **Document History**: Snapshot-based version history with restore features.
- **File Imports**: Upload and parse `.txt`, `.md`, and `.docx` into the editor via `mammoth`.
- **Export Tools**: Export current documents as Markdown or PDF.
- **SPA Routing**: Production-ready `vercel.json` for stable deep-linking on Vercel.
